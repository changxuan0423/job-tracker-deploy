# 求职申请管理

一个现代化的求职申请管理看板，支持看板视图和表格视图。

## 功能

- 📋 看板/表格双视图模式
- 🔍 搜索筛选与多维度排序
- ⏰ 截止日期倒计时与紧急提醒
- 📎 材料准备状态追踪
- 🔗 校招官网快捷入口（预置20家大厂）
- ✏️ 新增/编辑/删除申请

## 部署

纯前端单文件应用，直接打开 `index.html` 即可使用。

也可通过 GitHub Pages 部署：
1. 将此仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 保存后即可通过 `https://<username>.github.io/<repo-name>/` 访问
