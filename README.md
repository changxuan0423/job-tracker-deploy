# 求职申请管理

一个现代化的求职申请管理看板，支持看板视图和表格视图，适配桌面端与移动端。

## 功能

- 看板/表格双视图模式
- 搜索筛选与多维度排序（截止日期/更新时间/公司名/阶段）
- 截止日期倒计时与紧急提醒
- 材料准备状态追踪
- 校招官网快捷入口（预置20家大厂，支持自定义添加）
- 新增/编辑/删除申请
- 时间线详情视图
- 移动端完整适配（手机/平板/桌面三端响应式）

## 部署

纯前端单文件应用，直接打开 `index.html` 即可使用。

也可通过 GitHub Pages 部署：
1. 将此仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 保存后即可通过 `https://<username>.github.io/<repo-name>/` 访问

## 技术栈

- 纯 HTML + CSS + JavaScript，无任何外部依赖
- 响应式设计，三级断点适配（<400px / <768px / 769-1024px）
- CSS 变量体系，毛玻璃质感 UI
